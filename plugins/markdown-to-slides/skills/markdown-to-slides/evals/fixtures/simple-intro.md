# 새로 들어온 동료를 위한 워크스페이스 소개

> 첫 주에 알아야 할 것들

작성: 멘토 · 백엔드 팀

---

## 안녕하세요

이 워크스페이스는 백엔드 서비스 5개와 프론트엔드 3개가 모여 있는 모노레포예요.

- 첫 주에는 환경 셋업과 readme만 읽어도 충분합니다
- 2주차부터 실제 티켓을 잡습니다

---

## 1. 셋업

- `mise install` — 모든 도구가 한 번에 설치돼요
- `pnpm i` — 워크스페이스 의존성
- `task dev` — 로컬 서버 시작

처음 한 번만 하면 됩니다.

---

## 2. 일하는 방식

- 모든 PR은 `main`에서 분기해서 만들어요
- 커밋 메시지는 한국어 + Conventional Commits (`feat(scope): 한글 설명`)
- 리뷰는 2명 이상

---

## 3. 자주 쓰는 명령

- `task test` — 테스트 전체
- `task deploy:dev` — dev 환경 배포
- `cxdm env list` — 환경변수 점검
